# NVIM configuration

## Keybinds

| Mode   | Keybind         | Action                   | Description       |
| :----- | :--------       | :-------                 | :------------     |
| normal | `<leader>ne`    | `NvimTreeToggle<CR>`     | Open NvimTree     |
| visual | `<leader><Bar>` | `EasyAlign*<Bar><Enter>` | Format GHFM table |
